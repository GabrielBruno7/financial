# Financial Dashboard

## Objetivo
Este projeto foi criado como um MVP para facilitar o acompanhamento financeiro mensal, permitindo visualizar contas, movimentações e saldo consolidado em um único lugar.

<img width="1189" height="909" alt="image" src="https://github.com/user-attachments/assets/0fbb73cc-5a82-4f1c-b41c-43eddd10b62a" />


## Funcionalidades

- Cadastro de transações
- Registro de entradas e saídas
- Listagem de contas do mês
- Cálculo de saldo total
- Resumo de entradas e saídas
- Visualização de transações recentes
- Ocultação de valores na interface

## Tecnologias

**Backend:** Go, Gin, PostgreSQL, GORM, Docker  
**Frontend:** React, TypeScript, Vite, Tailwind CSS, shadcn/ui
